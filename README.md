# ✈️ TravelMate

A Full Stack Travel Management System built using React.js, Spring Boot, and MongoDB.

---

# 🚀 Features

✅ User Login & Register
✅ Dashboard with Greeting
✅ Dynamic Trip Planner
✅ Expense Tracker
✅ Insurance Tracking
✅ Destination Cards
✅ Responsive Sidebar
✅ Toast Notifications
✅ Mobile Responsive Design
✅ User Profile Avatar

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Vite
* CSS3
* React Router DOM

## Backend

* Spring Boot
* Java
* MongoDB
* JWT Authentication
